Hexo 後臺管理 (hexo-admin)

install hexo-admin

windows cmd

於 blog 資料夾下輸入指令
npm install hexo-admin --save //安裝 hexo-admin 文章管理的後台插件

start hexo-admin server

windows cmd

hexo-admin Open on browser
http://localhost:4000/admin/

Preview articles by myself

Open on browser
http://localhost:4000/

Stop hexo-admin Server

要將文章 Push 上 Github 之前要先停止本地伺服器。
STEP. 在命令提示字元的畫面中按下 Ctrl+C，然後再按一次 Ctrl+C 或輸入 Y 來停止。
